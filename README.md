## 🌸 Overview

**Auto Approve Bot** is a modern, open-source Telegram bot built with **Pyrogram** to simplify join request management for channels and groups. Designed with reliability, security, and ease of use in mind, it provides a clean and developer-friendly solution for Telegram community management.

> 💡 *Simple to deploy, easy to customize, and built for reliable long-term use.*

#### 🤔 Why Choose This Project?

- ⚡ **Save Time** — Reduce manual moderation by automating repetitive tasks.
- 🛡️ **Reliable** — Built with a clean, stable, and maintainable codebase.
- 🚀 **Easy Deployment** — Get up and running quickly on your preferred platform.
- 🔧 **Developer Friendly** — Well-structured project that's easy to modify and extend.

#### 🔄 How It Works

1. 📥 **User Sends a Join Request** — A user requests to join your Telegram channel or group.
2. ⚙️ **Bot Processes the Request** — The bot checks your configured settings and validation rules.
3. ✅ **Decision Is Applied** — If all conditions are met, the request is approved automatically; otherwise, it follows your configured behavior.
4. 🚀 **Community Stays Organized** — Join requests are handled consistently, reducing manual moderation and saving administrators time.

## ✨ Features

- 🤖 **Auto Approval** — Automatically approve Telegram join requests
- 👥 **Pending Requests** — Approve existing pending join requests
- 🔒 **Multi Force Subscribe** — Support multiple public & private channels
- 📢 **Broadcast System** — Send messages to all bot users
- 🛠️ **Maintenance Mode** — Temporarily disable bot services
- 🚫 **Ban & Unban Users** — Manage restricted users easily
- ⚙️ **Easy to Customize** — Clean and beginner-friendly source code
- 🚀 **Multi-Platform Deployment** — Deploy on Koyeb, Railway, Render, VPS & more
- 💬 **24/7 Developer Support** — Get help whenever you need it

## ⚙️ Configuration


| Variable | Description |
|:---------|:------------|
| `API_ID` | Telegram API ID |
| `API_HASH` | Telegram API Hash |
| `BOT_TOKEN` | Bot Token |
| `DB_URI` | MongoDB URI |
| `ADMIN` | Telegram User ID |
| `PICS` | Start Images |
| `IS_FSUB` | Enable / Disable Force Subscribe |
| `FSUB_EXPIRE` | Force Subscribe Expire Time |
| `AUTH_CHANNELS` | Force Subscribe Channels |
| `AUTH_REQ_CHANNELS` | Request FSUB Channels |
| `LOG_CHANNEL` | Log Channel |

## 🤖 Commands

```
start - Check Bot Alive.
stats - Check bot stats.
login - login to your telegram account for string session
logout - logout your telegram account
accept - Accept all pending requests from a channel or group.
ban - to ban a user.
unban - to unban a user.
banned - to check banned users.
maintenance - Toggle maintenance mode
broadcast - Broadcast messages to all bot users.
```
